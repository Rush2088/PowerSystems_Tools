# Power System Tools

A collection of engineering calculation tools for power system analysis.

## Tools

- **Fault Level Calculator** — Calculates three-phase and single-phase fault levels at the LV terminals of a distribution transformer, including inverter contribution.
- **Grid Source Impedance Calculator** — Derives positive- and zero-sequence Thévenin impedances from measured fault currents, for use in relay settings and network studies.
- **IEEE 1584-2018 Arc Flash Calculator** — Computes incident energy and arc flash boundary per the IEEE 1584-2018 model for LV and MV equipment.
- **Protection Coordination (TCC)** — Plots time-current characteristic curves for IEC/ANSI/IEEE overcurrent relays, transformer I²t withstand, and fault markers on a log-log chart.
