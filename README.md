# Power System Tools

A collection of engineering calculation tools for power system analysis.

**Live site:** https://rush2088.github.io/PowerSystems_Tools/

## Tools

- **Fault Level Calculator** — IEC 60909 short-circuit tool that calculates three-phase and single-phase fault levels at the LV terminals of a distribution transformer, with transformer K-factor and optional inverter contribution.
- **Grid Impedance Calculator** — Derives positive- and zero-sequence Thévenin source impedances from measured LLL and LG fault levels, for use in relay settings and network studies.
- **Arc Flash Calculator** — IEEE 1584-2018 incident energy, arc flash boundary, and PPE category for VCB, VCBB, HCB, VOA, and HOA equipment configurations across 208 V – 15 kV.
- **Protection Coordination** — Interactive overcurrent TCC plotter supporting IEC / ANSI / IEEE curves, highset stages, transformer I²t withstand, and grading margin table on a log-log chart.
- **Transformer R & X Calculator** — Derives series R and X from FAT nameplate Z% and load loss test data — outputs per-unit, HV-side (Ω), and LV-side (mΩ) values along with X/R ratio, SC power factor, rated currents, and turns ratio.
